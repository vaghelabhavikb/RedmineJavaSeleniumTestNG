Setup
------------------------------

1. Install Docker Desktop for Windows

2. Run this command:
	docker run -d --name redmine -p 3000:3000 redmine:latest

-----------------------------------
Enable API and generate key
-----------------------------------
1. Open Redmine in your browser: http://localhost:3000 (or http://localhost:PORT if you mapped a different port).

2. Login as admin / localadmin (or whatever credentials you set).

3. Top menu → Administration.

4. Click Settings → API tab.

5. Check Enable REST API (or REST API toggle).

6. Click Save.

Now generate API key:

1. While logged in as the user you want to use for tests (e.g., admin), click your username (top-right) → My account.

2. Scroll to API access key.

3. Click Show or Generate to reveal the key.

4. Copy that key — you’ll use it in X-Redmine-API-Key header

Restore the initial point for testing
--------------------------------------
If your current Redmine container is named redmine, run:

	docker commit redmine redmine_snapshot:v1

This creates a new Docker image that contains:

✔ all your settings
✔ plugins
✔ themes
✔ database data
✔ any modifications you made inside the container

Basically: a full frozen state of Redmine at this moment.

Restore it anytime:

Stop & remove current container:

	docker stop redmine
	docker rm redmine


Run from your snapshot:

	docker run -d --name redmine -p 3000:3000 redmine_snapshot:v1

Boom — your Redmine is restored exactly the way it was when you committed.
